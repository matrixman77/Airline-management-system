README.txt

All files contained in Stage 3 folder

Description:
Airline System that manages certain tasks such as passengers, booking, schedule, etc.
Compile all classes before compiling AirlineMenu.java.

Prerequisites.
JVM (Java Virtual Machine)
A java compiler
A terminal, command prompt, or interface
Optional: Any IDE with or without an compiler

Running Instructions without IDE:
1. Open the Command Prompt in the location that the .java files are stored or pathfind these files.
2. Run the .javac command by typing "javac filename.java" to create a .class file.
3. Then run the .java command by typing "java filename" to run the program.

Running Instructions with IDE:
1. Open filename.java with the respective IDE.
2. Use the compiler native to the IDE. 
NOTE: (If the IDE in use does not have a compiler use the first two steps from 'Running Instructions without IDE')
3. Use the excecute or run button to run the program.
